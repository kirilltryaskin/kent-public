## Тарифы Кент Агент

**Старт** — для знакомства
- Базовый лимит сообщений
- Долгосрочная память
- Работа по расписанию
- Обработка файлов и голосовых

**Про** — для активного использования
- Увеличенный лимит сообщений
- Все интеграции (умный дом, финансы, мониторинг)
- Приоритетная поддержка
- Создание сайтов и дашбордов

**Бизнес** — для команд и проектов
- Максимальный лимит
- Несколько агентов
- Кастомные интеграции
- Выделенная поддержка
